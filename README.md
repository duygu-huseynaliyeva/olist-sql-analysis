# Olist SQL Analysis

## Dataset

Dataset used: **Olist Brazilian E-commerce dataset**

It contains information about:

- Customers
- Orders
- Order items
- Payments
- Products
- Sellers
- Reviews
- Delivery information

- ## SQL Skills Used

During this project the following SQL concepts and techniques were used:

- SELECT statements
- INNER JOIN
- GROUP BY
- HAVING
- ORDER BY
- LIMIT

### Aggregate Functions
- COUNT()
- SUM()
- AVG()

### Subqueries
- Nested subqueries
- Correlated subqueries

### Common Table Expressions (CTE)
- WITH clause for complex queries

### Window Functions
- ROW_NUMBER()

### Conditional Logic
- CASE WHEN statements

### Filtering Techniques
- WHERE conditions
- EXISTS / NOT EXISTS

### Date Functions
- STRFTIME()
- JULIANDAY()

### Data Quality Analysis
- NULL value detection
- Missing data checks
